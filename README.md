\# HR Analytics Dashboard (Power BI)



\## Objective

Analyze workforce trends, attrition patterns, and employee demographics.



\## Tools Used

\- Power BI

\- Data Visualization



\## Dashboard Features

\- Employee Count KPI

\- Attrition Rate analysis

\- Salary and Age distribution

\- Department-wise insights



\## Key Insights

\- Identified high attrition in specific departments

\- Observed trends in employee age groups

\- Derived insights for HR decision-making
